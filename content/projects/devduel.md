Co-developed a website where players compete online in real time to solve LeetCode-style coding problems (e.g., sorting a list) head-to-head, owning the backend and game-data layer.

## Highlights
- Built the Java (Maven) REST backend, continuously reading/writing three Firestore collections (Users, Problems, Rooms) and streaming live game state to the front end
- Backed real-time lobbies, difficulty-based problem randomization, Clerk-authenticated and guest play, and a profile/leaderboard system
- Documented code and wrote implementation summaries, enabling seamless integration with the front-end team
- Shipped a fully playable version — authenticating logins, hosting lobbies, connecting users, and determining winners — with front-end and back-end test coverage across a 4-person team
