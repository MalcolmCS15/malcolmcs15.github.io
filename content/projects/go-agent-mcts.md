Built an AI to play the game Go on a 5x5 board, implementing the MCTS algorithm used in DeepMind's AlphaGo.

## Highlights
- Researched and implemented state-of-the-art algorithms adapted to a new setting, building on papers to maximize performance
- Implemented an A/B framework for testing optimizations like heuristics or time controls, achieving 3x faster testing
- Navigated practical constraints (strict 15-second time limit) and designed adaptive timing algorithms to improve results
- Made critical decisions on research-based features, like removing a CNN-based heuristic (using PyTorch) and a value net due to time constraints
- Performed well in a class-wide head-to-head tournament, finishing in the top 10 out of a pool of 150
